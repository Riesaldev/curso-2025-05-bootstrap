Guía Visual de Componentes Dinámicos de Bootstrap

Collapse

El componente collapse permite ocultar y mostrar contenido. Es ideal para secciones de FAQ, menús desplegables o cualquier contenido que deba ser accesible bajo demanda.

Ejemplo Visual:

Mostrar/Ocultar Contenido

Este es un contenido que se muestra y oculta con el componente collapse. Puedes poner cualquier tipo de contenido aquí, como texto, imágenes, o incluso otros componentes de Bootstrap.

Código HTML:

<p>
  <a class="btn btn-primary" data-bs-toggle="collapse" href="#collapseExample" role="button" aria-expanded="false" aria-controls="collapseExample">
    Mostrar/Ocultar Contenido
  </a>
</p>
<div class="collapse" id="collapseExample">
  <div class="card card-body">
    Este es un contenido que se muestra y oculta con el componente collapse. Puedes poner cualquier tipo de contenido aquí, como texto, imágenes, o incluso otros componentes de Bootstrap.
  </div>
</div>

Accordion

El acordeón es una extensión del componente collapse. Permite crear múltiples secciones colapsables, donde solo una puede estar abierta a la vez (o varias si se configura así).

Ejemplo Visual:

Este es el cuerpo del primer elemento del acordeón. Se muestra por defecto, gracias a la clase .show. El atributo data-bs-parent asegura que los demás colapsables se cierren cuando este se abre.

Este es el cuerpo del segundo elemento del acordeón. Está oculto por defecto y se expande al hacer clic.

Este es el cuerpo del tercer elemento del acordeón. También está oculto por defecto.

Código HTML:

<div class="accordion" id="accordionExample">
  <div class="accordion-item">
    <h2 class="accordion-header" id="headingOne">
      <button class="accordion-button" type="button" data-bs-toggle="collapse" data-bs-target="#collapseOne" aria-expanded="true" aria-controls="collapseOne">
        Elemento del Acordeón #1
      </button>
    </h2>
    <div id="collapseOne" class="accordion-collapse collapse show" aria-labelledby="headingOne" data-bs-parent="#accordionExample">
      <div class="accordion-body">
        <strong>Este es el cuerpo del primer elemento del acordeón.</strong> Se muestra por defecto, gracias a la clase <code>.show</code>. El atributo <code>data-bs-parent</code> asegura que los demás colapsables se cierren cuando este se abre.
      </div>
    </div>
  </div>
  <div class="accordion-item">
    <h2 class="accordion-header" id="headingTwo">
      <button class="accordion-button collapsed" type="button" data-bs-toggle="collapse" data-bs-target="#collapseTwo" aria-expanded="false" aria-controls="collapseTwo">
        Elemento del Acordeón #2
      </button>
    </h2>
    <div id="collapseTwo" class="accordion-collapse collapse" aria-labelledby="headingTwo" data-bs-parent="#accordionExample">
      <div class="accordion-body">
        <strong>Este es el cuerpo del segundo elemento del acordeón.</strong> Está oculto por defecto y se expande al hacer clic.
      </div>
    </div>
  </div>
  <div class="accordion-item">
    <h2 class="accordion-header" id="headingThree">
      <button class="accordion-button collapsed" type="button" data-bs-toggle="collapse" data-bs-target="#collapseThree" aria-expanded="false" aria-controls="collapseThree">
        Elemento del Acordeón #3
      </button>
    </h2>
    <div id="collapseThree" class="accordion-collapse collapse" aria-labelledby="headingThree" data-bs-parent="#accordionExample">
      <div class="accordion-body">
        <strong>Este es el cuerpo del tercer elemento del acordeón.</strong> También está oculto por defecto.
      </div>
    </div>
  </div>
</div>

Offcanvas

El componente Offcanvas permite crear barras laterales que se deslizan desde el borde de la pantalla. Es útil para menús de navegación, carritos de compra, o cualquier contenido que deba mostrarse temporalmente sin interrumpir el flujo principal de la página.

Ejemplo Visual:

Título del Offcanvas
Este es el contenido del offcanvas. Puede ser cualquier cosa, como un menú de navegación, información adicional, o un formulario.
  • Acción
  • Otra acción
  • Algo más aquí

Código HTML:

<button class="btn btn-primary" type="button" data-bs-toggle="offcanvas" data-bs-target="#offcanvasExample" aria-controls="offcanvasExample">
  Mostrar Offcanvas
</button>

<div class="offcanvas offcanvas-start" tabindex="-1" id="offcanvasExample" aria-labelledby="offcanvasExampleLabel">
  <div class="offcanvas-header">
    <h5 class="offcanvas-title" id="offcanvasExampleLabel">Título del Offcanvas</h5>
    <button type="button" class="btn-close text-reset" data-bs-dismiss="offcanvas" aria-label="Close"></button>
  </div>
  <div class="offcanvas-body">
    <div>
      Este es el contenido del offcanvas. Puede ser cualquier cosa, como un menú de navegación, información adicional, o un formulario.
    </div>
    <div class="dropdown mt-3">
      <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton" data-bs-toggle="dropdown">
        Menú Desplegable
      </button>
      <ul class="dropdown-menu" aria-labelledby="dropdownMenuButton">
        <li><a class="dropdown-item" href="#">Acción</a></li>
        <li><a class="dropdown-item" href="#">Otra acción</a></li>
        <li><a class="dropdown-item" href="#">Algo más aquí</a></li>
      </ul>
    </div>
  </div>
</div>

Carousel

El carrusel es un componente para mostrar una serie de contenidos (imágenes, texto, etc.) de forma rotativa. Es ideal para destacar productos, noticias o galerías de imágenes.

Ejemplo Visual:

Título de la Primera Imagen

Descripción breve para la primera imagen.

Título de la Segunda Imagen

Descripción breve para la segunda imagen.

Título de la Tercera Imagen

Descripción breve para la tercera imagen.

<div id="carouselExampleCaptions" class="carousel slide" data-bs-ride="carousel">
  <div class="carousel-indicators">
    <button type="button" data-bs-target="#carouselExampleCaptions" data-bs-slide-to="0" class="active" aria-current="true" aria-label="Slide 1"></button>
    <button type="button" data-bs-target="#carouselExampleCaptions" data-bs-slide-to="1" aria-label="Slide 2"></button>
